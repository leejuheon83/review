# 요구사항 요약 (MVP)

## 목표
- 팀장이 팀원 코칭 로그를 30~60초 내 기록
- 팀원별 히스토리 및 HR 분석/내보내기 제공

## 역할
- Manager: 본인 팀원 로그 CRUD
- HR(Admin): 전체 조회, 활동률/분포, Export

## 핵심 플로우
- 빠른 기록: 팀원 선택 → 유형 선택 → 한줄 메모(필수)
- 팀원 히스토리: 기간/유형/태그 필터, 검색, 수정/삭제
- HR: 팀장 활동률, 유형 분포, CSV Export

## 필수 제약
- memo 5~200자
- type enum 필수
- Manager는 본인 팀원 데이터만 접근
